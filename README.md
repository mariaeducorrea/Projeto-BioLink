# Projeto - Hub de Propostas & Conexões

Este projeto é o ponto central para captação de propostas e networking. Ele integra um formulário de contato inteligente com acesso direto às minhas principais redes profissionais e sociais.

## Tecnologias Utilizadas

* **HTML5** & **CSS3** (Interface responsiva)
* **JavaScript (ES6+)** (Lógica de validação e envio)
* **EmailJS API** (Integração de e-mail sem back-end próprio)

## Integração com EmailJS

O envio de e-mails é realizado através da API oficial do EmailJS.

* **Documentação utilizada:** [EmailJS REST API Guide](https://www.emailjs.com/docs/rest-api/send/)

## Configuração

Para rodar este projeto localmente, é necessário configurar as variáveis de ambiente (IDs do EmailJS) no arquivo de configuração:

* `SERVICE_ID`
* `TEMPLATE_ID`
* `PUBLIC_KEY`
